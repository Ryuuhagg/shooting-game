# シューティングゲーム

## 概要
C++とDxLibを使用して制作した2Dシューティングゲームです。
プレイヤーは自機を操作して敵を倒しながらスコアを獲得し、一定スコア到達後に出現するボスの撃破を目指します。

## 制作期間
約3カ月

## 開発環境
- C++
- DxLib
- Visual Studio 2022

## 操作方法

| 操作 | キー |
|--------|--------|
| 移動 | WASD / 矢印キー |
| 攻撃 | Space |

## 工夫した点

### 1. オブジェクトプールによる弾管理
敵弾やエフェクトを事前生成して再利用することで、生成・破棄による負荷を抑えました。

### 2. シーン管理システム
タイトル、ゲーム、リザルト画面をSceneクラスで管理し、シーン切り替えを容易にしました。

### 3. 演出の強化
ヒットストップや爆発エフェクト、スコアポップアップを実装し、攻撃時の爽快感を向上させました。

### 4. 敵行動のパターン化
直進、波移動、ジグザグ移動など複数の敵行動を実装し、ゲームプレイに変化を持たせました。

## ソースコードの見どころ

### EnemyManager
敵の生成や出現パターンの管理を行っています。

### BulletManager/EnemyBulletManager
プレイヤー弾・敵弾を管理しています。
オブジェクトプールを利用し、弾の生成と破棄による負荷を抑えています。

### ExplosionManager / HitSparkManager
爆発エフェクトやヒットエフェクトを管理しています。
プールによる再利用を行っています。
